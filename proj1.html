Celsius to Fahrenheit Converter

The JavaScript function on the home page uses the formula F = (9 / 5) × C + 32. It receives a Celsius number from the user and shows the Fahrenheit result.

This function was added to meet the assignment requirement for a new JavaScript function with user input.
